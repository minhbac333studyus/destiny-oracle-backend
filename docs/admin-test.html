List All Cards

GET /api/v1/cards

Returns all destiny cards for the current user in spread order.

Available Aspects

GET /api/v1/cards/aspects

All 10 built-in aspects with alreadyAdded: true/false for the current user.

Card Detail

GET /api/v1/cards/{cardId}

Add Aspect Card

POST /api/v1/cards

Add a new destiny card. Use a built-in aspectKey (e.g. "learning", "creativity") or set isCustom: true for a custom aspect.

Edit Card

PATCH /api/v1/cards/{cardId}

Update card fields. Only non-null fields are changed. aspectLabel / icon only allowed for custom aspects.

Remove Card

DELETE /api/v1/cards/{cardId}

⚠️ Permanently deletes the card and all its habits. Stage content and image history are preserved (DB cascade).

Toggle Habit Completion

PUT /api/v1/cards/{cardId}/habits/{habitId}/complete

Mark a habit as done or undone for today. Idempotent — toggling the same value twice has no effect.

Generate Image Prompts (Claude)

POST /api/v1/cards/{cardId}/generate-prompts

Calls Claude AI to generate 6 stage-specific image prompts. Requires ANTHROPIC_API_KEY env var. Returns prompts without generating images.

Generate All Stage Images (Gemini)

POST /api/v1/cards/{cardId}/generate-images

Full pipeline: Claude generates 6 prompts → Gemini Imagen generates 6 images in parallel (virtual threads) → stores in GCS → persists URLs in DB.

Requires ANTHROPIC_API_KEY + GOOGLE_CLOUD_PROJECT_ID. May take ~60s.

Regenerate Single Stage Image

POST /api/v1/cards/{cardId}/generate-images/{stage}

Regenerates one specific stage's image. Useful for re-rolling a result you don't like.

Generate Images for All Aspects

POST /api/v1/cards/generate-images/all

One button to generate card art for every aspect.

For each of the user's aspect cards (Health, Career, Finances…):
1. Claude writes 6 stage prompts (Storm → Legend)
2. Gemini Imagen generates 6 stage images in parallel
3. Images stored in GCS, DB updated

Aspects are processed one at a time (sequential) to avoid API rate limits. Each aspect gets its own GenerationJob — poll /{cardId}/jobs/latest per aspect.

⚠️ Requires ANTHROPIC_API_KEY + GOOGLE_CLOUD_PROJECT_ID. May take several minutes for all aspects.

Generation Job Tracker

GET /api/v1/cards/{cardId}/jobs/latest

Poll the pipeline status after calling Generate All Images. Shows all 12 steps (6 PROMPT + 6 IMAGE) with real-time status.

Job statuses: QUEUED → PROMPTING → IMAGING → COMPLETED | FAILED
Step statuses: WAITING → RUNNING → DONE | FAILED | SKIPPED

Set User Profile — Chibi & Avatar

PUT /api/v1/users/{userId}

Set the chibi_url and avatar_url for the current user.
chibi_url is used as a style reference image when Gemini Imagen generates card art — it gives the AI a visual anchor for the character's look.
Paste any public image URL (e.g. from Google Drive, Imgur, or your GCS bucket).
